# shURenZHOUluxun.github.io

Personal GitHub Pages site showcasing projects and small web games.

Live demo: https://shURenZHOUluxun.github.io

## Features
- Static portfolio site (HTML, CSS, JS)
- Small games under `games/` (e.g., Rock–Paper–Scissors)
- Responsive header and simple layout

## Quick start (local)
1. Clone the repo:
   ```bash
   git clone https://github.com/shURenZHOUluxun/shURenZHOUluxun.github.io.git
   cd shURenZHOUluxun.github.io
   ```
2. Preview locally (Mac):
   ```bash
   python3 -m http.server 8000
   # open http://localhost:8000 in your browser
   ```

Or simply open `index.html` in your browser.

## Project structure
- index.html — main page
- style.css — global styles
- games/ — game pages and scripts (e.g., rockPaperScissors.js)
- README.md — this file
